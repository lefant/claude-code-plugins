# Lefant Commands

This plugin no longer ships active slash commands from this repo.

The former Lefant command workflows have been migrated to `lefant/agent-skills`, where they are now available as skills across agent CLIs.

## Structure
- `.claude-plugin/plugin.json` — plugin manifest

## Usage
Use the skills from `lefant/agent-skills` instead:
- `architecture-decision-records`
- `changelog-fragments`
- `feature-specs`
- `atomically_land`
- `devlog`
- `handover`
- `recent_context_from_git`
- `git_resolve_merge_conflicts`
- `github_get_pr_comments`

This directory remains only as a migration marker in the Claude plugin repo.
