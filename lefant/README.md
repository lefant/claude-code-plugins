# Lefant Misc Commands

Collection of utility commands (currently `devlog` and `handover`) extracted from the monorepo for reuse as a plugin.

## Structure
- `.claude-plugin/plugin.json` — plugin manifest
- `commands/` — slash commands (`devlog.md`, `handover.md`)

## Usage
Install this plugin (or load via a local marketplace) to expose the commands in Claude Code sessions. No external dependencies required.
