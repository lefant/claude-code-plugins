# lefant-claude-code-plugins

Claude Code plugins for development workflows.

## Plugins

- **lefant** - Deprecated in this repo; the command workflows now live in `lefant/agent-skills`
- **ntfy** - Notifications via ntfy.sh
- **rpi** - Research/Plan/Implement workflow agents

## Related

Skills have been migrated to [lefant/agent-skills](https://github.com/lefant/agent-skills).

The `rpi` Claude plugin remains here because it still carries Claude-specific subagent configuration. A copy of the RPI workflow also exists in `agent-skills` for other CLIs.
